# Mercado Box Landing Starter

Готовое окружение для сборки интерактивного лендинга на **HTML + CSS + JavaScript**.

## Требования
- Node.js 18+

## Установка и запуск
```bash
npm install
npm run dev
```

## Сборка production
```bash
npm run build
npm run preview
```

## Структура
- `index.html` — разметка лендинга
- `src/style.css` — стили
- `src/main.js` — интерактив (кнопка + tilt-эффект карточки)
